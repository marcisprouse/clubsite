    • How to Add a Featured Flyer on the
    Home Page

    Background: Our Feature app within the Coyote Lakes Recreation Club website is managed in the administrative panel. When a feature is added to the back-end, it will appear on the home page of the site based on the Publish and Expire dates. The features are only visible if they are marked “Published.” They are ordered by “Published Date” with the feature Published Date closest to “now” appearing first. You can eliminate the feature from the home page by marking it as “Draft” as opposed to “Published” in the administrative panel.Additionally, you can set a date for the feature FLYER To expire, or disappear from the site.

    The featured event flyers are only seen by Members who are logged into the site. If you are a public person or not logged in, you will not see the featured event flyer.

    In order to create a feature on the home page, you will need to convert your flyer to an image (.jpg) file. You can do this a number of ways. You could take a screen shot of your flyer or take a good picture of it and crop it on your phone.

    There are online tools to convert .doc files to .jpg, too. This online tool works well: https://cloudconvert.com/docx-to-jpg

    Once you have your .jpg image of the flyer for the featured event, follow these instructions.

    Here is an example of what a feature flyer will look like on the home page of the site:

    Instructions:
    For the purpose of our example, we are going to add the feature, “Tex-Mex Dinner and Horse Race Game.”

    1. Login to the Administrative Panel
    2. Select Features under “PAGES”
    3. Choose “ADD FEATURE +” in the upper right-hand corner.
    4. Add the title of the featured event. In our example, we will add “Tex-Mex Dinner and Horse Race Game”. As you type in the Title, you will see the “Slug” automatically populates. Don’t worry about that field. It is there in the event we want to have separate pages for these flyers in the future.
    5. Next, add a description. You already have a lot of the description on the actual flyer. You may want to add links to RSVP here. If you go to the actual RSVP page of the event, you can copy the link from the URL bar of your browser and add it into the description. We are using a special way to add links in the description box. The formula looks like this: [Title of Link](https://www.....). If you write it like that, with the brackets for the title and the parenthesis for the link, right next to each other with no spaces, the link will appear correctly on the page.

      Example Link:
      [RSVP Here](https://www.coyotelakesrecreationclub.org/activity/respond/22)

      Add any encouraging, additional or other information that is not on the flyer. If our example case, we are going to write the following:

      Join us for some fun! Please be sure to
      [RSVP Here](https://www.coyotelakesrecreationclub.org/activity/respond/4). We can't wait to see you!

      Email Monica at [rlmhallmi@yahoo.com](mailto:rlmhallmi@yahoo.com)

    6. “Feature day time” field is the day/time that the event occurs. The Tex Mex dinner occurs on Saturday, September 4th at 3:00pm.
    7. Optionally, add a Feature Location. This is here just in case there is an event that takes place away from the club. It will appear under the date on the home page if it is filled out. For our example, we will leave it blank.
    8. Next, click on the “Choose File” button and select the .jpg file of the flyer for the featured event.

      When the file is uploaded, you will see the name of the file next to the “Choose File” button like this:

    9. The “Publish” field is important to consider because the events are ordered on the home page by the most recent “Publish” date, not the “Feature day time” date. If you want it to appear first, just use today as your Publish date. I like to use 12:00pm as the time. That time causes no issues with Time zones for us. Featured FLYERS will only appear on the site when it occurs BETWEEN the Publish date and the Expire date.
    10. The “Expire” field is set to when you would like the featured FLYER to disappear from the site. As mentioned before, featured FLYERS will only appear on the site when it occurs BETWEEN the Publish date and the Expire date. It will go away after this date whether it is set to "Published" or not. "Draft" will make the flyer go away whether the expired date has not occurred yet or not.
    11. Finally, the “Status” field is very important because it determines whether or not the featured event shows up on the page or not. You could set it up and come back later to mark it “Published”. Keep it as “Draft” to keep it off the home page. When you don’t want it on the home page anymore, you can either delete the feature or mark it as “Draft” again. We are going to mark it as “Published.” IT HAS TO BE MARKED “PUBLISHED” IN ORDER FOR IT TO APPEAR BETWEEN THE PUBLISH DATE AND THE EXPIRE DATE. IF IT IS MARKED “DRAFT”, IT DOESN’T MATTER IF THE PUBLISH DATE AND EXPIRE DATE IS SET, IT JUST WON’T SHOW UP.
    12. Now, check everything over and if OK, click “SAVE: at the bottom right-hand corner.
    13. Now you will see the record of it in the administrative panel.
    14. Go to the website and look to see if your featured flyer is there. It is! I used a lousy .jpg for this example, but you will use a much better one!

      You did it!!

      If you log out from the site, you will see that a person who is not logged in cannot see it.

      You can add as many features as you need to the home page, but it may look silly to have too many.

      You can control whether or not they are seen by marking them as “Draft” or “Published” in the back end. Be sure to have it marked “Published” if you are counting on it to appear based on the “Publish” and “Expire” dates. If it’s set to “Draft”, it will NOT appear, even between the “Publish” and “Expire” dates.
